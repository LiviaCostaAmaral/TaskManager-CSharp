# 📝 Task Manager em C#

Projeto desenvolvido em C# para gerenciamento de tarefas utilizando programação orientada a objetos e manipulação de arquivos.

## Funcionalidades

 Adicionar tarefas
 Listar tarefas
 Marcar tarefas como concluídas
 Remover tarefas
 Salvar tarefas automaticamente

## 🛠 Tecnologias

 C#
 .NET
 VS Code

## Estrutura

```text
TaskManager-CSharp/
├── Program.cs
├── Tarefa.cs
├── TarefaService.cs
├── README.md
```
## Como executar

Clone o projeto:

```bash
git clone URL_DO_REPOSITORIO
```

Entre na pasta:

```bash
cd TaskManager-CSharp
```

Execute:

```bash
dotnet run
```
